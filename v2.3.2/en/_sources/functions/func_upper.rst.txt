$upper
======

| Usage: **$upper(text)**
| Category: text

**Description:**

Returns ``text`` in upper case.


**Example:**

The following statement will return the value indicated::

    $upper(This text is UPPER case)  ==>  "THIS TEXT IS UPPER CASE"

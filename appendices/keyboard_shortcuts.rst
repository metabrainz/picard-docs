Main window
-----------

File
++++++

.. csv-table::
   :width: 100%
   :widths: 50 25 25
   :header: "**Action**", "**Windows / Linux**", "**macOS**"

   "Add folder", :kbd:`Ctrl+E`, :kbd:`⌘+E`
   "Add files", :kbd:`Ctrl+O`, :kbd:`⌘+O`
   "Save selected files", :kbd:`Ctrl+S`, :kbd:`⌘+S`
   "Quit Picard", :kbd:`Ctrl+Q`, :kbd:`⌘+Q`

++++++

.. csv-table::

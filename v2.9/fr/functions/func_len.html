$len

Utilisation: $len(texte)
Catégorie : texte

Description:

Renvoie le nombre de caractères dans texte.

Exemple:

Les instructions suivantes renverront les valeurs indiquées :

$set(foo,)
$len(%foo%)    ==>  "0"

$set(foo,ABC)
$len(%foo%)    ==>  "3"

$len()         ==>  "0"
$len(ABC)      ==>  "3"

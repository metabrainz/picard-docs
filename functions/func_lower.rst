$lower
======

| Usage: **$lower(text)**
| Category: text
| Implemented: Picard

**Description:**

Returns ``text`` in lower case.


**Example:**

The following statement will return the value indicated:

.. code-block:: taggerscript

    $title(tHe houR is upOn uS)  ==>  "the hour is upon us"

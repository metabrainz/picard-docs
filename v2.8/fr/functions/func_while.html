$while¶

Utilisation: $while(condition,code)
Catégorie: boucle
Mis en œuvre: Picard 2.3

Description:

Exécute le code à plusieurs reprises jusqu’à ce que la condition n’évalue plus à True. Pour chaque boucle, le décompte est stocké dans la variable _loop_count. Cela permet d’accéder à la valeur de comptage dans le script code.

Note

La fonction limite le nombre maximum d’itérations à 1000 pour éviter de créer accidentellement une boucle infinie.

Exemple:

L’instruction suivante définira return sur « Echo … echo … echo … »:

$set(return,Echo...)$while($lt(%_loop_count%,2),$set(return,%return% echo...))

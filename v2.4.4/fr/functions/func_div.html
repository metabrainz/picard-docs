$div¶

Utilisation: $div(x,y,*args)
Catégorie: mathématique

La description:

Divise x par y et renvoie la valeur entière (arrondie vers le bas). Peut être utilisé avec un nombre arbitraire d’arguments (par exemple: $div(x, y, z) = (x / y) / z). Si un argument est vide ou n’est pas un entier, la fonction renverra une chaîne vide. Si le deuxième argument ou tout argument suivant est zéro, la fonction renverra une chaîne vide.

Exemple:

Les instructions suivantes renverront les valeurs indiquées:

$div(10,3)      ==> "3"
$div(10,-3)     ==> "-4"
$div(-10,3)     ==> "-4"
$div(10,3,2)    ==> "1"
$div(10,-3,-2)  ==> "2"
$div(10,2,1.5)  ==> ""
$div(10,2,0)    ==> ""
$div(10,2,x)    ==> ""
$div(10,2,)     ==> ""
